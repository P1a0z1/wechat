 **漏洞名称** ：Windows DNS 服务器远程执行代码漏洞

 **漏洞出现时间** ：2023年7月11日

 **影响等级** ：严重

 **影响版本** ：

  * Windows Server 2012  

  * Windows Server 2008 

  * Windows Server 2019

  * Windows Server 2016

  * Windows Server 2022

 **漏洞说明：**  

    出现多个Windows DNS 服务器远程执行代码漏洞，但是目前microsoft以及cve暂时未公开详细说明。
